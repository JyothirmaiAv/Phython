
# Python Basics – Variables and Data Types

## Description
This file demonstrates basic Python concepts such as variables, data types, and type checking.

## Features
- String variable (name)
- Boolean variable (is_student)
- Integer variable (age)
- Float variable (score)
- Type checking using type() and isinstance()

## Technologies
- Python

## Learning Outcome
This exercise helped me understand Python data types and how to check variable types.

#Code

name = 'Alice'
print(name, type(name))

is_student = True
print(is_student, type(is_student))

age = 20
print(age, type(age))

score = 80.5
print(isinstance(score, float))

## Output
Alice <class 'str'>
True <class 'bool'>
20 <class 'int'>
True
